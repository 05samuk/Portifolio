Estrutura da Web

HTML é uma estrutura uma linguagem de marcação, ou seja, traves dela podemos criar um arquivo com a terminação .html e indentificar para o nosso navegador, que ele é um arquivo html, e podemos escrever um site através dele, no caso temos a nomenclatura o html semântico que nos possibilita deixar o código bem mais legivel, encaixar cada coisa no seu devido lugar e deixar o nosso site mais organizado.



CSS é uma linguagem de estilização, ela que vai dar mais cor ao nosso site, ou seja, através de uma boa estilizção podemos deixar um site muito mais atraente e deixando o usuário querendo ficar mais tempo no site, até porque ninguém fica em um site que não o agrada, o css tem esse trabalho quando bem estilizado pode chegar a resultados incriveis e a sites muitos lindos.



Javascript já esse e uma linguagem de programação esse é o cara responsável em deixar o seu site mais dinânico e usual ou seja aquele botão que abre e fecha o pop up quando misturados esses 3 juntos e capaz de criar coisas incriveis. Atraves dele temos tudo oque uma linguagem de programação nos oferece tipos primitivos funcões, variaveis e entre outros. Esta foi uma calculadora feita com a ajuda do professor André, onde funciona como a primeira calculadora porém nessa era para ter e funcionar as notações infixas, préfixa, e pósfixa. Além das notações nessa calculadora tembem tem os parenteses funcionais.

-Notação infixa é uma notação comum em aritmética e na lógica, onde os operadores são colocados entre os operandos em que eles atuam.

-Na notação pós-fixa os operadores aparecem após os operandos e não entre os operandos.

-A notação prefixa requer que todos os operadores precedam os dois operandos sobre os quais atuam.
