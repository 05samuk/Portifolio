Súmario

No primeiro semestre, aprendemos sobre automação industrial com projetos práticos. Usando as plataformas Tinkercad e Wookie, aprendi a criar e simular sistemas automatizados, ganhando uma boa noção prática desse campo essencial na tecnologia com a utilização da lingaugem C++ muito ultilazada para esse tipo de hardware.

Explicação

Através do arduino foi capaz de criar alguns projetos praticos colocando meu conhecimento a prova os links dos projetos serão deixados ao fim de cada explicação. Então vamos começar a explicar sobre os projetos, mas de forma resumida para podemos desenvolver os projetos precisamos entender sobre o basico de programação e estrutura de cada componente, no caso do arduino e do ESP32 temos void setup e void loop de forma resumida, as variaveis e as parte estáticas ficam no void setup e a lógica do código que vai se repetir fica no void loop e podemos criar funções auxiliares para ajudar no código.

PROJETOS

semaforo

Nesse projeto em c++, a sala teve que implementar seu conhecimentos para fazer e montar o arduino e fazer ele funcionar, nos foi proposto conectar leds e fazer elas brilharem igual a um semaforo onde começa piscando o led amarelo e depois o funcionamento padrão de um semaforo. Neste semaforo alem das leds do semaforo tem tambem as leds do semaforo dos pedestres onde o vermelho brilha quando o verde dos carros liga e vice versa.
