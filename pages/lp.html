Estrutura de condição

IF,ELSE E ELSE IF, são estrutras de condição ou seja aplicamos a lógica se uma coisa é verdadeira ela executa uma coisa se não ela executa outra, um exemplo para entender de forma rápida e simples é, existe uma casa de show onde apenas maiores de 18 anos podem entrar ou seja a logica que voce pode executar é, IF a idade for igual ou maior a 18 você deixa entrar, ELSE IF se for menor não pode entrar, e por fim para tratar um possivel erro, ELSE se a idade não for maior que 0 ou menor que 0.



SWITCH CASE, é uma outra estutura de condição, pense que quando você tiver usando muitos IF`s e ELSE`s está na hora de usar o SWITCH CASE ele evita a repetição de código e melhora bem a otimização um bom exemplo de uso é o seguinte, pensa que voce precisa verificar que dia da semana é, então voce pode usar um SWITCH CASE.

Orientação a Objetos

A orientação a objetos veio para deixar os códigos mais bonitos e de facil entendimento, mais perfomáticos. Dentro disso temos que seguir algumas regras no caso, os pilares bases da orientação a objeto, vou explicar de forma rápida e compreensivel cada uma:

1 - Encapsulamento: Agrupa dados e métodos em uma classe, controlando seu acesso e ocultando detalhes internos

2 - Herança: Permite que uma classe herde atributos e métodos de outra classe, promovendo a reutilização de código

3 - Polimorfismo: Um objeto pode ser tratado de várias formas, dependendo do contexto, aumentando a flexibilidade do código

4 - Abstração: Fornecer uma visão simplificada dos objetos, ignorando detalhes irrelevantes e concentrando-se no que é importante
